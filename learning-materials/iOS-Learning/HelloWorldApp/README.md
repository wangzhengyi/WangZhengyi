# iOS Hello World App

这是一个基于Swift的iOS Hello World项目，用于iOS学习和开发实践。

## 项目结构

```
HelloWorldApp/
├── HelloWorldApp/
│   ├── AppDelegate.swift          # 应用程序委托
│   ├── SceneDelegate.swift        # 场景委托（iOS 13+）
│   ├── Controllers/
│   │   └── ViewController.swift   # 主视图控制器
│   ├── Views/                     # 视图文件
│   ├── Models/                    # 数据模型
│   ├── Resources/                 # 资源文件
│   ├── Assets.xcassets/          # 图片资源
│   └── Info.plist                # 应用配置文件
├── HelloWorldApp.xcodeproj/      # Xcode项目文件
├── HelloWorldApp.xcworkspace/    # Xcode工作区文件
├── .gitignore                    # Git忽略文件配置
└── README.md                     # 项目说明文档
```

## 功能特性

- 基本的Hello World界面
- 简单的用户交互（按钮点击）
- 动画效果演示
- 标准的iOS应用架构

## 开发环境要求

- Xcode 12.0+
- iOS 12.0+
- Swift 5.0+

## 如何运行

1. 使用Xcode打开 `HelloWorldApp.xcodeproj` 文件
2. 选择目标设备或模拟器
3. 点击运行按钮（⌘+R）
