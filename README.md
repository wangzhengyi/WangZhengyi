- `resume/`：个人简历与求职相关材料；
- `blog/`：学习笔记与技术博客合集（含各平台与主题）。

## 使用说明
- 新增博客：在 `blog/` 目录下撰写新的 Markdown 文件，并在 `blog/README.md` 或本文件中增加索引。
- 更新简历：在 `resume/` 目录添加或更新对应简历文件，推荐命名为 `resume-<company-or-role>.md`。

仓库已移除示例工程与项目代码（如 `KMPDemo/`、`HelloWorldApp/`、各语言 `Algorithms` 代码目录、`life/` 等），仅保留文档与学习材料，保证整体简洁整洁。
